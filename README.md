# Learning Master

## Transform the Way You Learn

**Learning Master** is an intelligent, AI-powered learning management system designed to revolutionize how you study. Whether you're mastering new skills, preparing for exams, or organizing complex subjects, Learning Master adapts to your learning style and accelerates your progress.

### Why Learning Master?

- 🎯 **Learn Smarter, Not Harder** - AI-powered quizzes and adaptive learning paths personalized to your pace
- 📚 **All-in-One Learning Hub** - Notes, flashcards, quizzes, and study tools in one beautiful interface
- 🚀 **Boost Retention** - Spaced repetition and intelligent review systems maximize knowledge retention
- 📊 **Track Progress** - Real-time analytics and progress tracking keep you motivated and accountable
- 🤖 **AI Assistant** - Get instant explanations and personalized learning support

## ✨ Core Features

| Feature | Description |
|---------|-------------|
| **📝 Smart Note Editor** | Rich formatting editor for organizing study materials with intelligent structuring |
| **🎴 Flashcard Decks** | Create powerful flashcard sets with spaced repetition for optimal memory retention |
| **🧪 AI Quiz Generator** | Automatically generate challenging quizzes from your notes powered by advanced AI |
| **💬 Learning Chat** | Get real-time help, explanations, and personalized study guidance from your AI tutor |
| **📊 Progress Dashboard** | Track learning metrics, completion rates, and performance trends at a glance |
| **📋 Task Board** | Organize your learning goals and projects with an intuitive Kanban board |
| **📖 Learning Books** | Build comprehensive study materials and share knowledge with others |
| **⚙️ Learning Console** | Centralized management for all your educational resources and materials |

## 🚀 Getting Started

### Quickest Way: Docker

Start learning in seconds with a single command:

```bash
make up    # Launch Learning Master
```

The app opens at `http://localhost:3000`

**Stop & Restart:**
```bash
make down      # Stop the application
make restart   # Restart with one command
```
